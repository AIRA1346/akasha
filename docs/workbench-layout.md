## 2. 레이아웃 프로필 (2종)

### A. 대시보드 (작품 카탈로그)

| 모드 | 열 구성 (좌→우) |
|------|-----------------|
| **그리드만** (탭 없음) | 서재 \| (탭 접음) \| **카탈로그 그리드 전체** |
| **작품 열림** | 서재 \| 탭 \| **작품정보** \| **사용자 아카이브 (md)** |

```
그리드:  [서재][──── 카탈로그 ────]
열림:    [서재][탭][작품정보][████ md 메인 ████]
```

### B. 나만의 서재

| 모드 | 열 구성 (좌→우) |
|------|-----------------|
| **기본** | 서재 \| 탭(선택) \| **작품 전시 (포스터 그리드)** |
| **작품 탭 열림** | 서재 \| 탭 \| 작품정보 \| 사용자 아카이브 *(대시보드와 동일)* |

- 나만의 서재 **탐색**에는 **작품정보(3열) 없음** — 아카이브된 작품 카드 그리드가 메인
- 개별 작품을 탭으로 열면 md 편집·열람을 위해 **4열 작업 레이아웃**으로 전환

---

## 3. 열별 스펙

### 3.1 서재 레일 (1열)

- 기존 `DashboardSidebar` (~260px, 리사이즈 가능)
- 나만의 서재 / 대시보드 전환

### 3.2 탭 레일 (2열)

- 작품 1개 이상 열리면 **기본 표시**, **접기 가능** (`SharedPreferences`)
- 포스터 썸네일 또는 매체 아이콘 + 제목, `×` 닫기, dirty(●)
- 너비: 드래그 조절 + **고정(잠금)** 토글

### 3.3 작품정보 (3열) — 작품 열림 시 **항상 표시**

**편집 가능 (확정):** 포스터 · 별점 · 상태 · **제목** · **태그** · HoF · 작품/나의 상태

```
┌─────────────────┐
│   [포스터]      │  ← 이미지 교정
│   제목 (편집)   │
│   작가 · 연도   │  ← 사전 Fact (읽기 전용)
│   ★★★★★        │
│   상태 드롭다운 │
│   태그 (편집)   │
│   work_id 표시  │  ← 작품 정체성 (읽기 전용)
│ [기본값으로]    │  ← 초기화
│ [md 저장][삭제] │
└─────────────────┘
```

#### 제목 변경과 `work_id`

- **조인 키는 `work_id`** — Tier 1 사전·IP 1카드·프랜차이즈는 `work_id`로 유지
- YAML `title`은 **유저 표시명** (별칭·감상용 제목 변경 가능)
- **「기본값으로」** 버튼: `WorksRegistry` + 최초 아카이브 스냅샷에서  
  제목·태그·포스터(사전 기본) 등을 복원 ( `work_id`는 변경하지 않음)

### 3.4 사용자 아카이브 (4열) — **메인**

#### 4열에 무엇을 보여주나 (확정)

> **파일 원문 텍스트를 그대로 뿌리는 게 아님.**  
> md를 **읽기 좋게 렌더** (제목·인용·목록·이미지·표 등).

| 소스 | 역할 |
|------|------|
| `bodyRaw` | Obsidian에서 쓴 커스텀 섹션·본문 **보존** |
| `MarkdownBodyMerger` | 3열에서 저장하는 슬롯(시놉·명대사·메모)과 **충돌 없이 합침** |
| `VaultMarkdownBody` | 합쳐진 결과를 **화면에 예쁘게 표시** |

즉: **「md 내용을 앱 UI로 잘 보여준다」** + 저장 시 vault `.md`와 **동일한 내용**이 되도록 merge.

- Quick edit TextField **기본 UI 없음** — 4열 「본문」「.md」 탭에서 편집
- 3열 `md 저장` → YAML + merge된 본문을 vault에 기록

### 3.5 미아카이브 (사전-only, md 없음)

- 작품 탭 열기 → 3열 기본정보 편집 가능
- 4열: 안내 + **지금과 동일** — 편집 후 **`md 생성`** (`DetailArchiveSave`)
- 최초 저장 시 `.md` + `posters/` 템플릿 생성

| 열 | 기본 | 조절 |
|----|------|------|

## 8. 한 줄 요약

**대시보드:** 서재 → 탭 → 작품정보(전부 편집·work_id+초기화) → md 메인.  
**나만의 서재:** 서재 → 탭 → 포스터 그리드; 작품 열면 동일 4열 작업 뷰.  
열 너비는 드래그·고정 가능; 4열은 md를 렌더로 잘 보여 주며 저장 내용과 일치.
